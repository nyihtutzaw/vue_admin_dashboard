# vue admin dashboard

## Project Description
Plug and Play Dashboard build by Vuejs.

## Technology Usage
1. Vuejs
2. Materialzie
3. Sass
4. Vuex

## Project setup
```
npm install or yarn install
```

### Compiles and hot-reloads for development
```
npm run serve or yarn serve
```

### Compiles and minifies for production
```
npm run build or yarn build
```
